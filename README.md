# KeyFacturation

KeyFacturation surge desde la necesidad de realizar KPI's para el sector de digitalización, sector en el cual trabajo actualmente y,
una de mis tareas, es la realización de KPIs para medir el trabajo de las cargas y gestión de Prefacturación. Hace unas semanas, mi jefe, me solicitó
que realice un informe de KPI. Al realizarlos en word y no llevar registro histórico, esto me generó demasiados inconvenientes por lo que decidí
hacerlo de una manera mas sencilla, obteniendo todos los datos desde una aplicación llamada Sidom, generando un perfil que me permita visualizar
todos los datos necesarios y exportarlos a un Excel.

Se implementa también la gestión de facturación de honorarios y gastos en base a las cotizaciones existentes y permitiendo un cálculo exacto y preciso
de los montos a cobrar al cliente.

## Funcionalidades Principales

* Funcionalidad 1: Obtención de datos en un Data Grid view a través de la importación de un excel
* Funcionalidad 2: Seteo de informe en la base de datos
* Funcionalidad 3: Confección de PDF con KPI y de informes mensuales
* Funcionalidad 4: Calculos de honorarios precisos
* Funcionalidad 3: Informes de facturación.
